# opulence: 
  A Random Image Loader from a given Api with an adaptive background.

## Tasks:
- Fetches a random image from given Api
- Display the image
- Adapts the background color or design based on the image’s dominant colors — to make the interface visually cohesive
- Allow user to load another image on button click
- Handle error gracefully

### Dev Spec:
- Dart, Flutter
- Architectural pattern: MVVM, Clean Architecture
- State management: watch_it
- ServiceLocator (as DI)
